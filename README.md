# Syntra

Syntra is a Vite React + Express TypeScript demo app that turns Telegram customer-service messages into structured customer operations intelligence.

Run the full local demo stack:

```powershell
.\RUN_ALL.bat
```

That starts the frontend, API, and Telegram bot worker through `npm run dev:all`.

See [DEMO_README.md](DEMO_README.md) for environment setup, verification commands, and the hackathon runbook.
