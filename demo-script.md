# Syntra 2-Minute Demo Script

## 0:00-0:15

Open `/dashboard`.

Say: "Syntra turns Telegram customer chats into customer operations intelligence for a service business. This demo business is ApertureOne Events, an event photography company handling bookings, invoices, complaints, and delivery issues in Telegram."

Show active conversations, revenue at risk, urgent queue, and the AI Daily Brief.

## 0:15-0:35

Send this message to the Telegram bot:

```text
Hi, I booked a corporate event package last week but no one replied with the invoice. We need confirmation today or we'll find another vendor.
```

Say: "This is not a CRM form. It starts as an unstructured customer message."

If Telegram is unavailable, use Settings, Inject Demo Message, and say: "For demo reliability, this uses the same processing pipeline as Telegram ingestion."

## 0:35-0:55

Return to `/dashboard`.

Show the new urgent conversation and changed operational state: priority, sentiment, revenue risk, and suggested action.

Say: "The dashboard updates from the message, not from manual data entry."

## 0:55-1:20

Open `/inbox`.

Select the conversation. Show the Telegram-style thread, AI Intelligence panel, extracted fields, suggested next action, generated reply, linked tasks, confidence, and source evidence.

Say: "Every AI conclusion links back to the source conversation, so the operator can trust and inspect it."

## 1:20-1:40

Open `/pipeline`, then `/tasks`.

Show the lead or opportunity stage and the follow-up task. Click View Source Conversation on the task.

Say: "The same message created pipeline movement and work for the team. The task still carries its original Telegram evidence."

## 1:40-1:55

Open `/graph`.

Show the path from customer to conversation, task, owner, and risk. Click a node to show the inspector.

Say: "The operations graph makes the system different from a simple inbox. It maps conversation into business state."

## 1:55-2:00

Open `/settings`.

Show Telegram and OpenAI status panels.

Close with: "Syntra is not another chatbot wrapper. It converts customer communication into the live operating picture for a small business."
